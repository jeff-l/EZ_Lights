EZ_Lights



EZ_Lights provides a simple (“simple” being defined as my mom can use it) interface for remotely controlling lights and other household devices. The software will include

  • A simple user interface

  • Timers: where the software automatically turns on and off devices at specified times

  • Persistence: Configuration data like device names and timer data will be stored and used to re-configure the software on startup.

This project will be written in Java and will use the X10 protocol and X10 devices to control the household lamps and appliances. It's first implementation will focus on the X10 company's CM19A transmitter device which connects to the computer via the USB.

Although there are open source X10 projects, I am starting this one for the following reasons:

    • Very few support USB-based X10 devices

    • Few are written in Java which is the language I want to work with

    • Any that have UIs are not simple enough for my mom to use

    • Few support the CM19A

    • None do all the things I want (Java, USB-based interface, CM19A, simple UI)
